Eventful Escape

Take an in-depth look at the detailed code implementation of Avoid The Light by visiting this Repository link provided for the Project :)

Description

  • Find hidden keys to unlock challenges and progress through a chilling 3D horror experience. Dare to survive.

Features

  • Player Movement: Navigate the shadows as a bat using WASD controls! Dodge the lights and test your agility in this dynamic 2D game.
  • Insect: Insects, as food for the bat, spawn at random locations on the screen. Eat them to increase your score.
  • Level: Created a base level for combat that provides the immersive feel of being on a distinct reddish planet.

Implementation Details

  • Player Controller: Responsible for Player Movement using WASD Keys and Collision Detection.
  • Lights Controller: Handles the movement of lights on the screen, which the player must avoid.
