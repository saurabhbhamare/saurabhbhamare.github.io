Papyrus (Still Working)

Take an in-depth look at the detailed code implementation of Papyrus by visiting this Repository link provided for the Project :)

Description

  • 2D platformer game created using Unity. The game features puzzles that players must solve to unlock doors to the next levels. Players will encounter enemies like snakes and bees, which they must defeat to progress. Currently, the project is in the prototype stage, with all elements contained within a single frame.

Features

  • Player Movement: The player is equipped with dynamic abilities, including movement, jumping, and shooting bullets, offering a versatile and action-packed gameplay experience.
  • Sun's Redemption': A creative puzzle challenge where the sun is hidden behind shadowy clouds. Players must strategically move other clouds into the shadows, revealing the sun and unlocking the key to progress.
  • Level: Designed a visually unique level inspired by a paper theme, incorporating handcrafted aesthetics. Developed a wine set and crafted engaging challenges using base assets, blending creativity with functionality to enhance the gameplay experience.

Implementation Details

  • Service Handler: Responsible for the management of PlayerService, Enemy and UIService.
  • Player MVC: Implemented the player functionality using the MVC (Model-View-Controller) design pattern. This approach ensures a clean separation of concerns, with the model managing player data, the view handling graphical representation, and the controller coordinating interactions and gameplay logic for efficient and maintainable code.
  • Resource Pooling: Implemented resource pooling to manage player bullets, reducing memory usage and optimizing performance.
