Papyrus

This is a 2D platformer game created using Unity 2D. The game features puzzles that players
must solve to unlock doors to the next levels. Players will encounter enemies like snakes and bees



Implemented Service Locator Pattern

Managed player, enemy, UI services efficiently, improving game modularity and service accessibility..


Utilized MVC Architecture:

Ensured structured and maintainable player and inventory management, enhancing code
separation and making future updates straightforward.


Applied Object Pooling:

Enhanced game performance by reusing objects for handling player and enemy ammunition,
reducing memory usage and improving performance.
