Minesweeper

Take an in-depth look at the detailed code implementation of Minesweeper by visiting this Repository link provided for the Project :) GITHUB

Description

  • Minesweeper is a classic single-player puzzle computer game created using C++ and the SDL2 library. The objective is to clear a rectangular board containing hidden 'mines' or bombs without detonating any of them.

Features

  • Grid: 12x12 grid of Minesweeper tiles, enhanced with a sleek, modern design for the textures, offering a visually appealing experience crafted by CreatorModelIvesky.
  • Main Menu: Created a simple main menu with key input-based menu selection.
  • Sounds: Integrated sound effects for tile opening, mines, and flag placement, enhancing gameplay immersion. These were implemented using the SDL_Mixer library for seamless audio functionality.

Implementation Details

  • Window Class: Handles the window creation and initialization process.
  • Tile Class: This class handles the core logic for Minesweeper, including initializing the tile matrix, placing mines, and calculating adjacent mine counts. It also manages functionalities like flagging tiles, validating tile indices, and integrating sound effects using SDL_Mixer. The class ensures smooth gameplay through efficient algorithms and robust structure.
  • Render Class:This class manages the rendering and game state logic for Minesweeper. It initializes textures, handles various game states (main menu, gameplay, win, and lose), and renders tiles and numbers dynamically based on the game's progress. The class also includes flood-fill logic for revealing tiles, sound effects integration using SDL_Mixer, and efficient memory management for textures and resources.
  • Text and Sound Class: Handles SDL_Mixer for sound like mine click, tile opening, flag placement, TTF for text implementation in menus.
