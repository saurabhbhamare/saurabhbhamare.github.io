Description

  • This 3D FPS game, developed in Unity, is set on the alien planet Insectia. Players must navigate a hostile environment where insect enemies chase them, shoot bullets, and attempt to take them down. To survive, players must dodge incoming attacks, strategically shoot back, and outlast their adversaries in this thrilling battle for survival.

Features

  • Player Movement: In this game, Iâ€™ve implemented a unique movement feature that simulates gravity changes on the alien planet. This adds an immersive element to the gameplay, making exploration feel dynamic and otherworldly.
  • Enemies:Implemented two types of enemies: Larvae and Screecher. For Larvae, I utilized the AI NavMesh Follow feature to enable pathfinding, while for Screecher, I implemented functionality that allows it to shoot from the air at any angle.
  • Level: Created a base level for combat that provides the immersive feel of being on a distinct reddish planet.

Implementation Details

  • Main Handler: Responsible for managing essential resources and creating necessary services for the game. The resources include Scriptable Object references for players, Enemies. The services include PlayerService, BulletService, EnemyService and EventService.
  • Player Service: Developed the player using the MVC architecture, ensuring efficient separation of concerns and smooth functionality. Additionally, created and integrated the PlayerHUD into the screen, providing a clear and intuitive interface for enhanced user experience.
  • Enemy Service: Responsible for the creation of Larvaie and Screehcer.
  • Bullet Service: Handles Bullet implementation.
  • Dependency Injection: Implemented dependency injection in the main handler to efficiently manage services and provide the required dependencies.
  • Object Pooling: Used for the implementation of Player and Enemy bullets to optimize better game performamce.
